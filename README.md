# Purushoth M — Premium Portfolio

A colorful, animated, responsive static portfolio based on the supplied resume.

## Ready to publish
Upload `index.html`, `style.css`, `script.js`, and the `assets` folder to the root of a public GitHub repository.

Then:
1. Repository → Settings → Pages
2. Build and deployment → Deploy from a branch
3. Branch: `main`
4. Folder: `/ (root)`
5. Save
